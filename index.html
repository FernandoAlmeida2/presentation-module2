Webpack Asset Management

Fernando José de Almeida

1   O que são Loaders do Webpack?

  • São transformações aplicadas no código fonte de um módulo
  • Permitem gerenciar arquivos que não são JavaScript
  • Transforma esses diferentes arquivos em módulos válidos que o webpack consegue compreender
  • Podemos assim quebrar todo o código para melhor organização

2   CSS Loader e Style Loader

  • Loaders que permitem o uso de código css por meio de imports 
  • $ npm install -D css-loader style-loader
  • module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ["style-loader", "css-loader"],
      },
    ],
  },
};
  • import "./styles/styles.css"

3   O que é o Image Loader e como usá-lo?

  • O webpack 5 introduziu os "Asset Modules"
  • São módulos embutidos que permitem o uso de imagens, fontes, etc
  • Não é necessário instalar loaders adicionais 
  • {
  test: /\.(png|svg|jpg|jpeg|gif)$/i,
  type: 'asset/resource',
},
  • import MyImage from './my-image.png'
